# Travelwise

Travelwise is a simple and intuitive budgeting app designed to help travelers manage their expenses while on a trip. Users can create trips, organize expenses into categories, and track their total spending in real time.

## Features

- **Trip Creation**
  - Create and manage multiple trips.
  
- **Budget Categories**
  - Built-in travel expense categories including:
    - Food & Drinks
    - Transportation
    - Sightseeing
    - Flights
    - Hotels

- **Custom Categories**
  - Users can create their own categories to better fit their travel needs.

- **Expense Tracking**
  - Add expenses to specific categories within a trip.

- **Dynamic Spending Updates**
  - The total spending for a trip automatically updates whenever a new expense is added.

## How It Works

1. Create a new trip.
2. Use default categories or create custom ones.
3. Add expenses under the appropriate category.
4. Travelwise automatically updates the total spending for the trip.

## Use Cases

- Track travel expenses during vacations.
- Stay within a travel budget.
- Organize spending by category for better financial visibility.
